# System Zarządzania Wydarzeniami

Projekt realizowany w ramach przedmiotu **Modelowanie i Analiza Systemów Informatycznych**.

## Opis projektu

Celem projektu jest zaprojektowanie systemu informatycznego umożliwiającego zarządzanie wydarzeniami, takimi jak konferencje, szkolenia czy koncerty.

System umożliwia:
- tworzenie wydarzeń,
- rejestrację użytkowników,
- zapisywanie się na wydarzenia,
- zarządzanie uczestnikami,
- obsługę płatności.

## Zakres (Kamień Milowy I)

W ramach pierwszego etapu wykonano:
- analizę systemu,
- identyfikację aktorów,
- diagram przypadków użycia,
- wymagania funkcjonalne i niefunkcjonalne,
- wstępny diagram klas,
- diagram komponentów,
- diagram warstw systemu.
